Technical Specifications

Built on the Model Context Protocol standard with Rust performance and TypeScript flexibility.

Protocol
Model Context Protocol (MCP) v1.0
Backend Language
Rust (Axum framework) + TypeScript SDK
Transport
HTTP/2, WebSocket, stdio
Authentication
API Key, OAuth 2.0, mTLS
Data Sources
Filesystem (smart-crate.toml), Sled KV, SurrealDB, Supabase
Default Port
15195 (MCP Server)
Compatible Clients
Claude Desktop, Cursor, Cline, any MCP client
Performance
<10ms response time, 1000+ req/sec

Quick Start

Get up and running in under 5 minutes

1

Install the MCP Server

# Clone and build
cargo
install ctas7-mcp-server
2

Configure Claude Desktop

// ~/.cursor/mcp.json
"ctas7": {
"command": "ctas7-mcp-server",
"args": ["--port", "15195"]
}
3

Start Using It

Ask Claude: "List all CTAS-7 smart crates with their QA scores"
